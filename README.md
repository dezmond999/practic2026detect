# practic2026detect
Эксплуатационная практика 2026 год. Задача - детектинг типа объекта на фотографии с высота полёта дрона.
Программа, которая смотрит на фото с дрона, находит все машины, показывает, где какая, и ещё подсказывает, какой дрон лучше подойдёт для съёмки. Всё это в окошке с кнопками, зумом и подсказками при наведении.

**Что умеет:**
- Находить машины на аэроснимках (легковушки, грузовики, автобусы, фургоны, мотоциклы)
- Работать с большими фото — разбивает их на куски, чтобы ничего не потерять
- Показывать статистику — сколько и каких машин нашёл
- Искать в своей базе знаний — по запросу или по найденным типам машин
- Приближать, отдалять, таскать картинку, наводить на машину — появляется подсказка

---

## Как это устроено

Программа состоит из нескольких частей:

1. **Детекция** — модель YOLOv8, которую я дообучил на аэроснимках
2. **Тайлинг** — разбиение фото на куски, чтобы не сжимать его и не терять мелкие машины
3. **База знаний** — список из 6 дронов с их характеристиками (время полёта, для каких машин подходят)
4. **Поиск по смыслу** — через эмбеддинги и косинусное сходство (не по ключевым словам, а по смыслу)
5. **Интерфейс** — на tkinter, с зумом, панорамированием и подсказками

**Вот как это выглядит в структуре:**

- `detection.py` — 643 строчки, основная программа
- `best.pt` — дообученная модель (дообучил через Google Colab)
- `dataset` — датасет из вручную собранных 100 фотографий
- `itog_sukho_detect.py` — абсолютно сухой код со строгим выполнением задания, без визуала

## Откуда брал данные

Я использовал два источника:

- **Свои фото** — наделал 100 снимков с дрона в разных местах (парковки, трассы, дворы)
- **VisDrone** — открытый датасет, 10 000+ аэроснимков, уже размеченных

**Классы, которые определяет модель:**

| Что находит | Как называется | Цвет рамки |
|-------------|----------------|------------|
| Легковая машина | car | зелёный |
| Фургон | van | светло-зелёный |
| Грузовик | truck | оранжевый |
| Автобус | bus | красный |
| Мотоцикл | motor | пурпурный |
